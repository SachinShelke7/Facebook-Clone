# Facebook-clone

*Live Link https://facebook-clone-psi-seven.vercel.app/

## Features

The features include:

* 📝 Facebook Clone.
* 📡 Realtime database From Firebase.
* 📦 Full responsive
* 📡 Hosted on Vercel.

## Used
* Next.js,
* Tailwind CSS,
* Next Auth,
* Firebase
